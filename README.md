# QR-Code-Scanner

## How to setup?

1.    Clone the repo.
2.    Run ````npm install && cd ios && pod install````.
3.    Setup RNFirebase.
      - [Android](https://rnfirebase.io/#2-android-setup)
      - [iOS](https://rnfirebase.io/#3-ios-setup)
4.    Run ````npx react-native start````.
